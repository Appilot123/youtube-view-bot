Installation & quickstart

Minimum requirements: Node 18+ or Python 3.10+, Chrome/Chromium, Git, and a proxies provider. Docker optional.


# Clone
git clone https://github.com/username/youtube-view-bot.git
cd youtube-view-bot

# Option A: Node runner
npm install
npm run setup
npm run start

# Option B: Python runner
python -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
python run.py

# Option C: Docker
docker build -t youtube-view-bot .
docker run --rm -it \
  -e PROXY_URL=file:/data/proxies.txt \
  -v $(pwd)/config:/app/config \
  youtube-view-bot

Configuration & safety best practices

Proxy & fingerprint isolation

Assign a unique proxy and browser fingerprint per account. Prefer residential/ISP proxies. Persist cookies and storage per profile to reduce re-verification.

Rate limiting & cooldowns

Throttle actions, randomize schedules, and stagger logins. Implement per-account and global caps.
